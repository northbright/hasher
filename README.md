# hasher

hasher is a [Golang](https://golang.org) package provides functions to compute hash checksums. It's based on [iocopy](https://github.com/northbright/iocopy).

## Supported Hash Functions
* `MD5`
* `SHA-1`
* `SHA-256`
* `SHA-512`
* `CRC-32`
